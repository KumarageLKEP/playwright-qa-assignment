# Playwright QA Automation Assignment

## 📌 Overview
This project is a Playwright + TypeScript automation framework built using the Page Object Model (POM) design pattern.

It covers:
- Authentication
- Product catalog
- Cart functionality
- Checkout flow
- Cross-browser testing

---

## ⚙️ Prerequisites
- Node.js (v16 or higher)
- npm installed

---

## 🚀 Installation

```bash
npm install
npx playwright install
